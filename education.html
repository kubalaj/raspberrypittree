KEY TERMS

In the world of high preformance scientific computing as well as in this website there are a multitude of terms that will help you better understand what is happening.

CLUSTER

A computer cluster consists of a set of loosely connected or tightly connected computers that work together so that in many respects they can be viewed as a single system.(from Wikipedia)

A cluster can be thought of as hooking together computers with some sort of network cable to enable them to run some problems faster. A lot of the time when computing large data sets it makes more sense for data to be ran on more than just one computer. In this case there is the cluster.
